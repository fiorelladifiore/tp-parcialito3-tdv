# TP - Técnicas de Documentación y Validación: API REST de GitHub

Proyecto Java (Spring Boot) que documenta y utiliza un subconjunto de la
[API REST oficial de GitHub](https://docs.github.com/en/rest), cubriendo los 5 incisos del TP:

1. Autenticación con Bearer Token (nunca hardcodeado).
2. Listado de repositorios propios.
3. Creación de un repositorio.
4. Creación de un Pull Request.
5. Comentario en un Pull Request.
